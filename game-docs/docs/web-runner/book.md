# 书架表

### book
负责管理书架上的书籍信息

|   字段    |   说明 |
| :------  |   :------ |
| id         | 书架配置唯一ID |
| recommend       | 是否为推荐书籍,1是0不是 |
| display         | 是否展示,1是0不是 |
| name         | 图书的名字 |
| index         | 图书显示顺序 |
| net         | 网络配置ID|
| icon         | 图书的icon |
| tag1         | 图书的标签1 |
| tag2         | 图书的标签2 |
| tag3         | 图书的标签3 |
| gold         | 销售金币价格 |


### net
负责管理书架相关的网络配置信息

|   字段    |   说明 |
| :------  |   :------ |
| id         | 网络配置唯一ID |
| server       | 书架资源服务器地址|


- 关于图书封面Icon资源url完整的路径为: **net.server**+**book.icon**
